# Item Key

## Asset Naming Convention

All item assets follow a structured naming pattern:

### Navigation Items (Categories)
These items open a new radial dial with sub-items:
- `item_1.png` - Power Modules
- `item_2.png` - Metallic Alloys
- `item_3.png` - Synthetic Lubricants
- `item_4.png` - Electrical Components
- `item_5.png` - Mechanical Parts
- `item_6.png` - Optical Systems

## File Format Requirements
- Format: PNG with transparency
- Recommended size: 128x128 or 256x256 pixels
- Keep file size under 100KB per image
